Per-panel plan

Panels ride flat, longer dimension along the truck length · a row holds panels end-to-end up to the max-per-row cap · the gap before the last panel in a row uses the "2 panels" value; every other gap uses the "2+ panels" value · earlier-install panels always sit above later ones in the same truck · trucks on site can be pulled from simultaneously; the oldest is dispatched when a new one is needed.
